# BMI Calculator 🧮⚖️

A simple and user-friendly BMI (Body Mass Index) Calculator developed using Python. This application helps users calculate their BMI based on height and weight inputs and provides health status classifications according to standard BMI ranges.

## 📌 About the Project

Body Mass Index (BMI) is a widely used measure to determine whether a person's weight is appropriate for their height. This project automates the BMI calculation process and displays the corresponding health category. BMI is commonly calculated using the formula: weight (kg) divided by height (m) squared. :contentReference[oaicite:0]{index=0}

## 📖 How to Use

1. Launch the application.
2. Enter your height.
3. Enter your weight.
4. Click the **Calculate BMI** button.
5. View your BMI value and health category instantly.

## 📊 BMI Categories

| BMI Range | Category |
|------------|------------|
| Below 18.5 | Underweight |
| 18.5 – 24.9 | Normal Weight |
| 25.0 – 29.9 | Overweight |
| 30.0 and Above | Obese |

These categories are commonly used in BMI calculators and health applications. :contentReference[oaicite:1]{index=1}
